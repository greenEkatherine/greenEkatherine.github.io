CrossLingo

Учи английские слова, создавая и разгадывая кроссворды по темам

Скриншоты

Описание

Кроссворды - это простой и увлекательный способ изучения иностранных слов. Для удобства слова разбиты по темам, по ним можно генерировать огромное множество уникальных кроссвордов. В начале необходимо указать свой уровень владения английским языком, чтобы использовать для кроссвордов слова заданной сложности. После этого уровень можно изменить в настройках. Удобная подсветка правильных ответов сразу поможет себя проверить. В игре можно воспользоваться подсказкой для буквы или слова, если не знаешь ответа. Такая подсказка стоит определенное количество звезд, которые начисляются за правильно отгаданный кроссворд. Пробуй, решай, тренируй память и пополняй словарный запас!

Что внутри:

  • выбор слов по заданным темам (одной или нескольким)
  • разная сложность для разных уровней владения языком
  • создание неограниченного количества кроссвордов
  • портретный режим ориентации экрана
  • подсказки для букв и слов
  • работа без интернета
  • отсутствие рекламы
